open-log: an open-source online data logger project.

open-log

An open source tool for uploading, storing, and viewing data feeds (typically, time series data) online, based on the Flask web microframework.

open-log is currently at a very rudimentary stage. Feeds are currently stored as simple csv files. We've created an example feed with feedID=3 in order to play with some of the initial features.

Please feel free to fork the code on github, play with it, and add to it!

Historical plot

A feed's history can be plotted via the python matplotlib plotting library by using "http://www.open-logger.pvos.org/data/FEEDID>/plot", where "FEEDID" is the number of the feed. E.g., for feed 3, you'd use "http://www.open-logger.pvos.org/data3>/plot". You can try that for feedID=3, here.
Historical data

A feed's history can be downloaded as a ".csv" file, by navigating to "http://www.open-logger.pvos.org/data/FEEDID/csv".

You can try this for feed 3, here
Most recent data point

The latest uploaded data point in a feed can be retrieved as JSON or as CSV.
  • For JSON, use: "http://www.open-logger.pvos.org/data/FEEDID/latest/json" (try that here for feedId=3).
  • For CSV, use: "http://www.open-logger.pvos.org/data/FEEDID/latest/csv" (try that here for feedId=3).
Uploading data

A feed can be updated via "http://127.0.0.1:5000/senddata?feedID=FIELDID&secretKey=SECRETKEY&field1=FIELD1&val1=VAL1".

You'll need a secret key to do this :)

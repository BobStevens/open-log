open-log: an open-source online data logger

Using FEEDID=3, SOMEVALUE (a floating point number), and SECRETKEY (you have to have a secret key already!) ...

Try navigating to:

http://www.open-logger.pvos.org/senddata?feedID=FEEDID&secretKey=SECRETKEY&field1=temp&val1=SOMEVALUE

And then to:

http://www.open-logger.pvos.org/data/FEEDID/plot

You can see all the data here: http://www.open-logger.pvos.org/data/FEEDID/csv

And you can get the latest data point here (JSON):

http://www.open-logger.pvos.org/data/FEEDID/csv

and here (CSV):

http://www.open-logger.pvos.org/data/FEEDID>csv
Source code is on github here.
